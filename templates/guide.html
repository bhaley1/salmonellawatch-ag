What this dashboard shows

SalmonellaWatch-Ag displays SNP clusters from NCBI Pathogen Detection that have at least one non-human isolate (food, animal, or environmental) with a collection date in the last 180 days. Each cluster card summarizes the genomic, epidemiological, and commodity context of that cluster.

The agricultural focus means this dashboard surfaces clusters relevant to pre-harvest food safety, environmental monitoring, and commodity-source tracking — including clusters that may not yet have associated human illness.

Using the filters

Country filter

Restricts the view to clusters with recent non-human isolates from the selected country. When a country is selected, human case counts and isolate lists within each card are also filtered to that country.

Serotype filter

The dropdown lists every serotype present among active clusters, sorted alphabetically. Two special options appear at the top:

  • ⚠ Serotypes of Concern (SOC) only — filters to the 21 serotypes identified by Katz et al. (2024) as high priority for reducing salmonellosis risk. SOC serotypes are marked with ⚠ SOC throughout the dashboard.

ST filter

Type a sequence type number (e.g., 152 or ST152) to show only clusters with that MLST type. MLST was run on one representative assembly per cluster using the Achtman 7-gene scheme for Salmonella enterica.

Anatomy of a cluster card

Each cluster card contains these sections, top to bottom:

Header row

SALMONELLA serotype Infantis ST32 PDS000027464.84 — the NCBI SNP cluster accession and version. On the right: human case counts for the 180d, 30d, and 15d windows.

Signal chips

Colored badges below the header flag noteworthy epidemiological signals. Click any chip to open its definition on the Methods page in a new tab. See the chip reference table below.

Typing strip

Shows the consensus serotype, MLST sequence type, cluster age (span), cumulative isolate counts, and collection-to-deposit lag.

Source profile (last 180d)

Badges showing how many recent non-human isolates fall into each commodity category: Poultry 557 Bovine 9 Swine 6 Water 485 with a total count.

Metadata summary

One-line summary: cluster age, total isolates, human/nonhuman split, ST, and countries represented.

Reading the histograms

Each card has two stacked-bar charts:

  • Collection frequency by year — the full history of the cluster, showing growth over time
  • Collection frequency — last 12 months — monthly resolution for recent trends

Bars are color-coded by commodity source:

Human Poultry Bovine Swine Produce Water Other
Tip: Compare the histograms with the ACCELERATING chip. If a cluster is flagged as accelerating, the yearly histogram should show a visible uptick in recent years. If it doesn't, the signal may reflect a batch deposition rather than real-world acceleration.

Collapsible sections

Click the ▶ triangles to expand detailed data within each card:

  • Poultry Isolates (N), Bovine Isolates (N), etc. — individual isolate records with PDT accession links to NCBI, collection dates, and geographic origin
  • Human cases (N) — recent human clinical isolates in the cluster
  • Geographic footprint — SVG world map showing where isolates have been collected. Red dots = human, teal = non-human. Dot size scales with count; opacity fades with age.

Example workflows

1. "What Infantis clusters are active in the U.S.?"

Set Country → United States, Serotype → Infantis. The banner shows the count. Sort by Acceleration rate to see the fastest-growing clusters first. Check the source profile badges to identify which commodities are involved.

2. "Are any clusters transitioning from food to human?"

Set Sort → 🔺 Emerging only. This filters to clusters with the EMERGING chip — those where the human-case fraction has recently increased significantly above the historical baseline. These represent the highest-priority signals for food-safety intervention.

3. "Show me all SOC serotype clusters with poultry isolates"

Set Serotype → ⚠ Serotypes of Concern (SOC) only. Scan the source profile badges for Poultry counts. Expand the Poultry Isolates section for individual sample details.

4. "What ST is associated with this serotype?"

Select a serotype from the dropdown. The ST appears in each cluster's header (e.g., ST32) and in the typing strip. Use the ST filter to cross-reference: type the ST number to find all clusters of that sequence type regardless of serotype.

5. "Is this cluster internationally distributed?"

Look for MULTI HEMISPHERE or MULTI CONTINENT chips. Expand Geographic footprint to see the map. Check whether non-human isolates come from different countries than the human cases (look for the IMPORT SIGNAL chip).

Key limitations

  • Genomic relatedness ≠ outbreak. Clusters are computational predictions. Confirming a foodborne link requires epidemiological investigation.
  • Data lags real-time by hours to days. The ingest runs nightly from NCBI.
  • Commodity classification is keyword-based. Isolates with vague or missing isolation_source fields may be classified as "Other."
  • Not all isolates have collection dates. Undated isolates are excluded from time-based filters and histograms.
  • MLST covers active clusters only. Clusters outside the 180-day window may show "pending" for their sequence type.
  • This is a personal research project. It is not an official USDA, CDC, or FDA surveillance system. Do not use for regulatory decisions without independent verification.
